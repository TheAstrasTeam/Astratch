# AstraEditor-Next——Astratch

> The next-generation Scratch 3.0 editor – faster, more modern, and more powerful.

---

## Introduction

**AstraEditor-Next** is a modern editor fully compatible with the [Scratch 3.0](https://scratch.mit.edu/) project format.  
It is rebuilt with cutting‑edge frontend technologies, and on the basis of retaining Scratch's original ease of use and ecosystem compatibility, it greatly improves performance, scalability, and development experience.

Whether for educational scenarios, creative programming, or secondary development, AstraEditor-Next is committed to being a better choice.

---

## Key Features

-  **Full Scratch 3.0 Compatibility** – Directly open/save `.sb3` projects, seamlessly connected with the official ecosystem.
-  **Performance Leap** – Uses more efficient rendering engines and state management to significantly reduce latency and memory usage.
-  **Modular Architecture** – Core functionalities are decoupled, making it easy to customize and extend, convenient for secondary development.
-  **Modern UI** – Responsive design with dark theme support, more in line with current aesthetics.
-  **Developer‑Friendly** – Clear type definitions, comprehensive documentation, and a hot‑reload development environment.
-  **Cross‑Platform** – Based on web technologies, supports all modern browsers, and can be packaged as a desktop application.
