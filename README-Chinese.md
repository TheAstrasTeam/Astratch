# AstraEditor-Next——Astratch

> 下一代 Scratch 3.0 编辑器 —— 更快、更现代、更强大。

---

## 📖 简介

**AstraEditor-Next** 是一个完全兼容 [Scratch 3.0](https://scratch.mit.edu/) 项目格式的现代化编辑器。  
它基于前沿前端技术栈重构，在保留 Scratch 原有易用性和生态兼容性的基础上，大幅提升性能、可扩展性和开发体验。

无论是教育场景、创意编程，还是二次开发，AstraEditor-Next 都致力于成为更优的选择。

---

## ✨ 主要特性

-  **完全兼容 Scratch 3.0** – 直接打开/保存 `.sb3` 项目，与官方生态无缝衔接。
-  **性能飞跃** – 使用更高效的渲染引擎和状态管理，显著降低延迟和内存占用。
-  **模块化架构** – 核心功能解耦，易于定制和扩展，方便二次开发。
-  **现代化 UI** – 响应式设计，支持暗色主题，更符合当下审美。
-  **开发者友好** – 清晰的类型定义、完善的文档和热重载开发环境。
-  **跨平台** – 基于 Web 技术，支持所有现代浏览器，并可打包为桌面应用。
